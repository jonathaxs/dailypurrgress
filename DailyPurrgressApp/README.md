# DailyPurrgress (Xcode Project)

This folder contains the original Xcode project version of **DailyPurrgress**.

The app was primarily developed, tested, and refined in this format using Xcode before being adapted to a Swift Playgrounds package for submission to the Swift Student Challenge.

## What’s inside

- Full `.xcodeproj` project
- Complete source structure (Models, Views, Components)
- Original asset catalog and resources
- Development and debugging environment

## Note

The Swift Playgrounds submission version is located in:

`DailyPurrgressPlaygrounds/App/DailyPurrgress.swiftpm`

If you want to explore the full development workflow with Xcode, open:

`DailyPurrgress.xcodeproj`
